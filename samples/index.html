Developer Samples

After setting up xmla4js on your system, you can start using these samples:

  • Synchronous examples:
    • discover-schema-rowsets.html - A basic example showing how to use Xmla.js to do synchronous retrieval of OLAP metadata. This sample uses various flavours of the Discover method and renders the results as HTML tables.
    • discover-synchronous.html - The sample lets you enter a URL for a XML/A webservice, and then visualizes the metadata in a dynamic treeview, allowing you to hierarchically explore datasources, catalogs, cubes, dimensions, hierarchies, levels, members and properties. This sample illustrates how to apply Restrictions to the XML/A Discover method.
    • rowset-examples.html - Xmla4js offers great flexibility in the ways to obtain data from the Xmla provider. The goal is to make it as easy as possible to directly use that data for third party visualization libraries and the like, with no or little hand-written processing and/or iteration. This sample illustrates various methods for iterating through and extracting data from a Xmla.Rowset
    • execute-synchronous.html - This sample illustrates how to select a XML/A datasource, send MDX queries, and retrieve and traverse the multidimensional resultset. This sample uses synchronous requests.
    • execute-synchronous.html - This reproduces issue 6.
  • Asynchronous examples
    • discover-asynchronous.html - This sample illustrates how to get XML/A metadata asynchronously.
    • execute-asynchronous.html - This sample illustrates how to select a XML/A datasource, send MDX queries, and retrieve and traverse the multidimensional resultset. This sample uses synchronous requests.
  • Combination Examples
    • xmlash.html - XML/A shell. Because real men use the command line.
    • yui-chart-wizard.html - This sample illustrates how to render a YUI chart using data obtained from an XML/A execute request.
    • pivot-table.html - This sample illustrates how to handle raw XML response and apply an XSLT stylesheet to it to render a pivot table.
    • treeview-pivottable-combination.html - This sample illustrates how to render 3 axes of a multidimensional data set in a pivot table / treeview combination. Leaf nodes in the pivot table represent tuples in the 3rd query axis, and provides a selection mechanism to display a corresponding pivot table.
